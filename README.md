# 🌍 Language Translator

A multilingual web application that translates text between different languages in real time. The application also supports speech input and speech output to improve user accessibility and experience.

## 🚀 Features

- 🌐 Translate text into multiple languages
- 🎤 Speech-to-Text
- 🔊 Text-to-Speech
- 📋 Copy translated text
- 📝 Translation history
- 📱 Responsive design
- ⚡ Fast translation using API

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Translation API
- Web Speech API

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/NikhileshKhairnar/Language-Translator.git
```

Open the project folder.

Run `index.html` in your browser.
